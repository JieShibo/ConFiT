# ConFiT
Source code of "Alleviating Representational Shift for Continual Fine-tuning" (CVPR-W 2022)
### Requirements
+ Python 3.9.1
+ PyTorch 1.10.0
+ TorchVision 0.11.1
+ [Timm](https://rwightman.github.io/pytorch-image-models/) 0.4.12
+ [Avalanche](https://avalanche.continualai.org/) 0.0.1 
### Run
```sh 
sh run.sh
```
